# Terraform AWS Web Platform

A Terraform platform for running a containerised app on AWS — VPC, ECS Fargate, RDS, and an ALB, with a GitHub Actions CI/CD pipeline.

Work in progress — building this module by module to learn Terraform and AWS platform engineering properly.

## Status
- [ ] Networking (VPC, subnets)
- [ ] Security groups
- [ ] Load balancer
- [ ] Database (RDS)
- [ ] ECS Fargate service
- [ ] CI/CD pipeline
